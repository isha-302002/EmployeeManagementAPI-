# Employee Management API

ASP.NET Core Web API project implementing CRUD operations for employee management using SQL Server and Entity Framework Core.

## Technologies Used
- ASP.NET Core Web API
- C#
- SQL Server
- Entity Framework Core
- Swagger UI

## Features
- Add Employee
- Get Employees
- Update Employee
- Delete Employee

## API Endpoints
- GET /api/Employee
- POST /api/Employee
- PUT /api/Employee/{id}
- DELETE /api/Employee/{id}

## Tools Used
- Visual Studio 2022
- SQL Server
- GitHub
